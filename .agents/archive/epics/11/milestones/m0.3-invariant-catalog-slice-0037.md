## Authority and State Impact

No production authority, runtime state, endpoint contract, transport behavior, or UI behavior changed. This slice adds documentation and a documentation-backed architecture regression.

## Compatibility Impact

No runtime compatibility changed. The only new failure mode is local backend test failure if the invariant catalog disappears, loses required columns, or leaves required metadata empty.

Revert the catalog section, the architecture-test catalog guard, and the milestone/capability documentation updates. Existing Slice 0036 M0.3 skeleton checks remain independently runnable.
