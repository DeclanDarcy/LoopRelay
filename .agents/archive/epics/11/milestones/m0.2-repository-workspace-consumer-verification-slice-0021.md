# Milestone 0.2 Slice 0021: Repository Workspace Consumer Verification

Date: 2026-06-26

## Objective

Extend the repository workspace Contract Oracle pilot from fixture-only coverage to downstream consumer verification using the same verifier family already used by the repository dashboard pilot.

## Scope

Contract identity: Repository workspace.

Projection: `RepositoryWorkspaceProjection`.

Authority: backend-owned projection shape after backend HTTP JSON serialization.

Consumers verified:

- Rust shell `RepositoryWorkspaceProjection` mirror.
- Manual TypeScript `RepositoryWorkspaceProjection` type.
- Dev Tauri mock `get_repository_workspace` command payload.

- `tests/CommandCenter.Backend.Tests/ContractFixtures/repository-workspace.golden.json` remains the Oracle-observed backend fixture for the workspace response shape.
