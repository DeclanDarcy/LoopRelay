## Architectural Impact

This slice formalizes the existing separation between:

- backend Oracle fixture truth,
- recursive consumer comparison,
- source-specific downstream shape providers,
- behavior tests that assert current consumer status.

It does not create a production contract API and does not expand the Oracle beyond the repository dashboard pilot. The dev mock provider remains intentionally narrow and source-shape-specific.
