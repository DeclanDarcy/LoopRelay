## Local-vs-CI Consistency Report

Local verification baseline exists and is executable.

CI verification baseline does not exist. Repository inspection found no `.github` directory, so there is no current GitHub Actions workflow to compare against local commands.

Certification impact: Milestone 0.1 can certify local structural verification only after this gap is either repaired by adding CI or quarantined with owner, risk, and retirement criteria. It cannot claim local-vs-CI consistency while CI is absent.
