## Known Limits

- This slice formalizes and guards drift metadata. It does not yet implement source scans for each drift class.
- Frontend regression area, shell command-family classification, regression UX lifecycle, confidence model, and M0.3 certification remain pending.
- Drift severity is modeled now, but several classes still begin as future inventory or planned enforcement until their owning milestones add executable scans.

## Rollback

Revert the drift model table, the backend guard, and milestone/capability updates. The prior M0.3 invariant catalog, taxonomy, ownership, and severity model remain valid, but future regressions would lose the shared architectural drift vocabulary until replaced.
